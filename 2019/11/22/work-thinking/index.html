关于工作的思考

发表于 2019-11-22 | 分类于 talk

今天带我的师傅突然问起工作以来做了哪些事情，有关转正的事情，才意识到时间已经偷偷溜走小半年，不禁有一点唏嘘，好像每次都在感慨时间过得太匆忙（：

不过这半年的运气真的太好啦，遇到了世界上最最最最好的女朋友，每天的心情就好像山上的小鹿，蹦来蹦去，只要她一句话就可以开心起来，想要做她最坚强的后盾，想要把故事认真地写成我们。感谢她走进我的人生~

因为自己不再是一个人啦（我不做人啦，JOJO），所以要静下心思考和她以后一起走的道路。她给了我好多好多勇气，让自己能够更加有信心，不想再选择消极的逃避。

自己内心还是非常喜欢做技术的，目前做的可能不是自己想做的事情，一方面改编译器以及写 clang static analyzer 对自己的难度还是有点大的，不过还是想要努力先做好这个，在下班时间自己学习自己喜欢的东西，先沉淀几年。在做好这个可能会再去挑战下一件事情。

自己也明白自己技术成长最快速的一段日子，平均每天大几百行的代码产出，不过工作之后也并非没有时间，只是自己很多都浪费在没必要的地方。

关于工作要做的改变：

  • 积极的态度
  • 想清楚解决问题的道路
  • 善于求助别人
  • 写小工具来完成重复的工作
  • 更清晰的沟通

自己的学习方面：

  • 不要给自己设限太多
  • 一万小时定律
  • 好好学习英语
  • 深入几项自己的拿手技术

生活方面：

  • 积极锻炼身体
  • 和她一起体会更多的人生
  • 偶尔抽时间读技术无关的书籍
  • 好好休息
  • 珍惜时间

今后的路我们一起走，希望我们都可以变得更好~
